# Multi-fiber array (MFA)-based photometry system
This page provides resources for the publication on MFA development

## grids
The folder contains three .stl files and a Python-based FreeCAD macro.

## fabrication
There is a PDF file detailing MFA fabrication procedures.

## MFA image processing workflow
The MATLAB LiveScript is the workflow for processing a .cxd file to analyse photometry signals.
